# GestureVoice 🤖🖐️

**GestureVoice** is an AI-powered web application that converts simple hand gestures into text and speech using a webcam. The project aims to improve communication accessibility by enabling users to express common messages through hand gestures that are instantly recognized and spoken aloud.

---

## 🌟 Features

* 📷 Real-time webcam-based gesture recognition
* 🖐️ Detects common hand gestures
* 📝 Converts gestures into text
* 🔊 Speaks the recognized text using Text-to-Speech
* 💻 Runs entirely in the browser
* 📱 Responsive and modern user interface
* ⚡ No special hardware, gloves, or sensors required

---

## 🎯 Purpose

Many people with speech or hearing impairments rely on sign language for communication. However, not everyone understands sign language.

GestureVoice bridges this communication gap by translating simple hand gestures into spoken words, making everyday interactions easier and more inclusive.

---

## ✨ Supported Gestures

| Gesture         | Output  |
| --------------- | ------- |
| 👋 Open Palm    | Hi      |
| 👍 Thumbs Up    | Okay    |
| ✌️ Victory Sign | Yes     |
| ✊ Closed Fist   | No      |
| 👌 OK Sign      | Perfect |
| ✋ Raised Hand   | Stop    |

More gestures and complete Indian Sign Language (ISL) support can be added in future versions.

---

## 🚀 How It Works

1. Open the website.
2. Allow webcam access.
3. Show one of the supported hand gestures.
4. The system detects the gesture in real time.
5. The gesture is converted into text.
6. The browser speaks the recognized word automatically.
